# HikkInn — Product Vision

## Problem

Northern Pakistan (Gilgit-Baltistan, Khyber Pakhtunkhwa highlands) is a
fast-growing tourism destination, but the hospitality & travel sector still runs
on **manual, phone/agent/walk-in systems**:

- Budget guest houses & jeep rentals are reachable only by phone/WhatsApp.
- No centralized digital inventory for stays, meals, transport, or activities.
- Flights are weather-dependent with multi-day cancellations and no live
  rebooking.
- Adventure products (paragliding, heli tours, ATV rental) have weak discovery
  and unverified operators.

This project was **awarded** to upgrade these manual systems into an automated,
AI-assisted, web-hosted combined-services platform.

## Two-sided platform

### 1. Admin side (property & service operators)
Manage everything the tourist can book:
- **Properties**: guest houses, hotels, resorts — rooms, amenities, seasons,
  pricing, availability.
- **Services**: tours, flights, car/vehicle charters, adventure activities.
- **Food**: menu, per-day / per-meal prepackaged plans, kitchen capacity.
- **Fleet / drivers**: vehicles, driver companion-app onboarding, live
  availability.
- **Ops**: bookings, orders, dispatch board, payouts, reports.

### 2. Tourist / end-user side
Customize an entire trip in one place:
- **Stay**: pick property + room type + dates.
- **Meals**: per-day, per-meal selection (not "breakfast included"); prepackaged
  plans available; live menu.
- **Transport**: on-demand, Uber-style taxi/vehicle hire with live driver
  availability; charter vehicle classes (ATV, Cessna, Helicopter).
- **Activities**: wind gliding, parachuting, skydiving, rafting, treks, etc.
- **AI customization**: an assistant builds a personalized itinerary from live
  inventory (rooms, meals, transport ETAs, charter slots).

## North-Pakistan centric

Initial focus regions: Hunza, Skardu, Swat, Naran-Kaghan, Chitral, Kumrat,
Gilgit, Astore, Fairy Meadows. Domestic tourists (Lahore/Karachi/Islamabad)
are the primary volume; international (Chinese, European) is the high-value
upsell.

## Success criteria
- A tourist can book room + per-meal plan + live taxi + a charter activity in a
  single session.
- An operator can move from paper/phone to full digital inventory & dispatch.
- Live dispatch (food + taxi) works like Uber — order placed, driver in
  vicinity notified.
