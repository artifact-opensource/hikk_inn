# HikkInn — Charter Booking (Aircraft + ATV)

Asset-light charter marketplace for high-value, low-frequency products:
helicopters, Cessna-class fixed-wing, and ground excursions (ATV, etc.).

## Model (Blade Air Mobility analogue)
Blade is an asset-light platform: it does **not** own aircraft but partners with
vetted operators; sells **by-the-seat**, **full charter**, and **crowdsourced
empty-seat fill**. HikkInn follows the same shape.

### Product types
| Type | Example | Booking modes |
|---|---|---|
| Helicopter | Skardu–K2 scenic, valley hops | seat / full / fill |
| Fixed-wing (Cessna-class) | Islamabad–Skardu charter, sightseeing | seat / full / fill |
| ATV / ground | trail excursions, glacier access | hourly / daily rental |
| Other aircraft | future eVTOL / seaplane | seat / full |

### Booking modes
- **By-the-seat**: book an individual seat on a scheduled/recurring flight.
- **Full charter**: reserve the whole aircraft/vehicle for a slot.
- **Crowdsource fill**: operator opens empty seats on a full charter; other
  tourists can buy remainder.

## Operator onboarding
- Vetted operators only (safety certs, licence, insurance).
- Define routes/zones, schedule slots, seat & full prices, capacity.

## Integration with live dispatch
- Helicopter/Cessna seat availability can surface in the tourist app next to
  taxi/activities; full charters get a dedicated booking flow.
- ATV rentals reuse the vehicle/fleet + driver model (with or without driver).
